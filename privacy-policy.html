Privacy Policy

1. Overview

We take your privacy very seriously. This privacy policy details how our browser extension collects, uses, stores, and protects your personal information. By using our product, you agree to the terms of this privacy policy.

2. Permission Usage

Our extension requires the following permissions to provide a complete functional experience:

Storage Permission

Used to save user-customized pomodoro times, task lists, and extension settings for future use.

Notifications Permission

Used to display task and pomodoro timer reminder notifications.

Offscreen Permission

Used to continue running pomodoro timer, countdown, and task reminder functions when the extension is not open.

SidePanel Permission

Used to display task lists and pomodoro timer in the browser sidebar, allowing users to view and manage tasks at any time.

Alarms Permission

Used for custom pomodoro timer reminder functionality.

ActiveTab Permission

Used to remember the last window position and size for convenient restoration to the same position and size next time.
